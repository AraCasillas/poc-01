# Proxy as an intermediary [Proof of concept]

In recent months, we have seen a rise in phishing sites that are exploiting tools such as Rallway and others, which feature legitimate authentication tools, most frequently using Microsoft tools. 

In this proof of concept, I will demonstrate how it is possible to use these tools and features to obtain credentials without arousing suspicion. To do this, I will use Ruby and an open-source tool similar to Rallway, together with artificial intelligence (to set up the logging and Coolfy configurations), thereby showing just how easy it is to do this without much technical knowledge, relying solely on ‘vibe coding’.
